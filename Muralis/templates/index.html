MURALIS - API orcamento domestico

Depesas do mes vigente = /api/despesas

Despesas totais = /api/despesasTotais

Adicionar:

Nova despesa = /api/despesas/add/despesa

Nova categoria = /api/despesas/add/categoria

Novo tipo de pagamento = /api/despesas/add/pagamento

Mostrar:

Despesas por categoria = /api/despesas/<nomeDaCategoria>

Todas as categoria cadastradas = /api/categoria

Todos os tipos de pagamentos cadastrados = /api/pagamento
